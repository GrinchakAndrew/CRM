    Sold Lead:
    • Scenario : If the lead was sold and user clicked on the GREEN button:
      When : Lead has been sold successfully
      And : Sales Manager clicked on the Green button
      Then : PopUp opens with data to fill in per lead, where:
      - additional email can be added \ edited
      - Sale_representative or sale_manager can assign agent via dropdown and select the date and time of the meeting
      - it is possible to leave a comment for user agent
      - Pay options can be selected:

      - Credit card,
      - cash or bank transfer

      If selected credit card:
      Display drop-down with:
      Crediting Company,
      number card,
      Expired date, CVV Code, Personal id of israel.
      It is possible to select partial payment (on an installment plan).
      If partial payment selected with checkbox
      User can enter number of first payment and date of the Payment of the remaining amount.
      If user selected payment partial:
      if the amount is too large, pre-payment can be made and activate the checkbox for remaining sum payment date.
      If the amount is too large, and the client wants to pay in bits and pieces,
      then the checkbox is to be selected for payment in parts,
      and from the drop-down menu client can select the partial breakdown of the payment ( from 1 - to 12).
